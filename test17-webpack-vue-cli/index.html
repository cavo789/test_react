test17-webpack-vue-cli

Using Bulma CSS framework

Remarks:

  • If you don't see anything, please, in a command prompt, go in folder `test17-webpack-vue-cli` and run `npm run serve` that will start http://localhost:8081/,
  • If you try to access to http://localhost:8081/ and it doesn't work, it means that you've not yet run `npm run serve`,
  • If you modify a source like the css, js or html and don't see changes, run `npm run build` to see the change in the browser.
